
============================================= S M A R T   P A R K I N G ===================================================

Bienvenu sur le programme de parking intelligent, afin de bien lancer le programme veuillez suivre les étapes suivantes : 

	1. Création de base de données : 
		Créer la base de données à partir du fichier "parking.sql" qui existe sur le dossier courant (importer le fichier).

	2. Lancer le MAIN_CONTAINER avec le fichier/class : PrincipalContainer.java

	3. Lancer l'agent AR avec le conteneur : ARContainer.java

	4. Sélectionner la région souhaitée, après vous allez que les agents parking vont être créer automatiquement.

	5. Lancer l'agent AB avec le conteneur : ContainerAB.java.

	6. Voir la communication sur le sniffer 

	7. Et voilà, le résultat de votre demande de réservation va être affiché.







PS. : Veuillez trouver dans le dossier libraries les EXTERNAL JARS dont vous aurez besoin.
